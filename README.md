# 🍽️ Gourmet Plate Restaurant Website

A complete restaurant management system with online ordering, table reservations, and payment processing.

## ✨ Features
- 👤 User registration & authentication
- 📋 Interactive menu with categories
- 🛒 Shopping cart system
- 💳 Secure payment processing
- 📅 Table reservation system
- 📦 Order history tracking
- 📱 Responsive design

## 🛠️ Technologies Used
- **Backend**: PHP 7.4+, MySQL
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: MySQL with foreign key constraints
- **Security**: Password hashing, SQL injection prevention
- **Storage**: Session-based cart, LocalStorage for payment
